                           Command line options


==============================================================================
Options

    --jframe       Use a JFrame instead of terminal.
                   (This will be the best option on a Windows machine.)

    --simple       Just a single window and a minibuffer. No "integrated".
                   output window. With this option it looks a bit more like
                   Vim.

    --init=<path>  Where <path> is the path to a file. Use this to load a clj
                   file on startup. If there is a .liq file it will not be
                   loaded.
